# 경조사 - 경조사의 Luma

경조사 일정 관리를 위한 모바일 퍼스트 웹앱입니다. [Luma](https://lu.ma) 스타일로 경조사(결혼, 부고, 돌잔치, 생일, 집들이)를 한눈에 관리합니다.

## 주요 기능

### 📅 참가자 뷰
- 예정된 경조사 **타임라인** 형태로 조회
- 유형별 필터링 (결혼/부고/돌잔치/생일/집들이)
- D-day 카운트다운
- **RSVP 응답** (참석/불참)
- 축의금/조의금 금액 기록 (빠른 입력 지원)

### 🏗️ 호스트 뷰
- **RSVP 현황 대시보드**: 초대/확정/미응답 한눈에
- **하객 관리**: 상태별 필터, 그룹별 리마인드 추천
- **비용 최적화 시뮬레이터**: 보수적/균형/공격적 시나리오 비교

### 💰 축의금 내역
- 올해 지출 총액, 참석 횟수, 평균 금액 통계
- 경조사별 지출 기록

### 👤 프로필
- 참석/호스트 이력 통계
- 알림 설정, 연락처 연동

## PWA 지원

- 모바일 퍼스트 반응형 디자인
